# مشروع تخرج: بوابة مجتمع حائل (Hail Community Portal)

مشروع متكامل لربط مقدمي الخدمات المحلية بالمستفيدين في منطقة حائل، تم تطويره كمتطلب لنيل درجة البكالوريوس في علوم الحاسب والمعلومات - جامعة حائل.

---

## عن المشروع (About the Project)
بوابة مجتمع حائل هي منصة إلكترونية (Marketplace) تهدف إلى تسهيل الوصول للخدمات المهنية والتقنية داخل مدينة حائل. يوفر النظام بيئة آمنة للمستخدمين للبحث، الحجز، والتواصل المباشر مع الخبراء.

## التقنيات المستخدمة (Tech Stack)
تم استخدام أحدث التقنيات لضمان أداء عالٍ وتجربة مستخدم سلسة:
- **Frontend:** React.js + Vite (TypeScript)
- **Styling:** Tailwind CSS + shadcn/ui
- **Backend/Database:** Supabase (PostgreSQL, Auth, Storage, Real-time)
- **State Management:** React Hooks
- **Charts:** Recharts

---

## ميزات النظام (Core Features)

### 1. لوحة تحكم المستفيد (Client)
- استعراض الخدمات حسب التصنيفات.
- نظام حجز مواعيد ذكي (يمنع تعارض الحجوزات للمزود).
- محادثة فورية (Real-time Chat) مع إمكانية إرفاق صور.
- نظام تقييم ومراجعة الخدمات.

### 2. لوحة تحكم المزود (Provider)
- إدارة الخدمات (إضافة، تعديل، حذف).
- متابعة الطلبات وإحصائيات الأداء (رسوم بيانية).
- نظام اشتراكات وفترة تجريبية تلقائية (30 يوماً).
- تواصل مباشر مع العملاء لحل المشكلات.

### 3. لوحة تحكم المسؤول (Admin)
- مراقبة وتوثيق الخدمات الجديدة قبل نشرها.
- إدارة المستخدمين (توثيق الحسابات، حظر الحسابات المخالفة).
- نظام تذاكر الدعم الفني والشكاوى.
- مراقبة المحادثات لضمان جودة التعامل.
- تصدير تقارير العمليات بصيغة Excel/CSV.
